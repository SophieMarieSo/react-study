### ✍🏻 기능 정의

1. `left pane` 연락처를 등록할 수 있는 form
2. `right pane` 연락처 목록과 검색창
   - 연락처 목록에 이름과 전화번호를 추가할 수 있다
   - 사용자 이름으로 검색할 수 있다
   - 연락처를 삭제할 수 있다
   - 연락처를 즐겨찾기 할 수 있다
   - 즐겨찾기 된 연락처는 최상단에 표기된다
